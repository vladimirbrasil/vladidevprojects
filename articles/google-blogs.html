Google Blogs

Vladimir Dietrich · June 13, 2025 ·3 min read
Português|English|Français|Deutsch|Español

Imagine um Google Doc. Pronto: temos mais um entrada para um blog. “Google Blogs”!

A ideia deste projeto foi facilitar a manutenção de um blog.

Primeiro pensei qual a maneira mais fácil de gerar conteúdo?

Bem, em primeiríssimo lugar aparecem as redes sociais e até mesmo sites especializados como o Medium. Mas pouco a pouco estas primeiras opções foram inventando restrições: primeiro, o Medium começou a cobrar por postagens; após, facebook e instagram começaram a impedir textos muito longos.

Então precisei reformular a pergunta: qual a maneira mais fácil de gerar conteúdo com bastante liberdade?

Aqui o Google Docs se destaca. Ele foi desenhando e é mantido para ser uma das maneiras mais práticas e fáceis de gerar conteúdo. Posso ditar o texto pelo celular, por exemplo. Pronto: tenho mais um Google Doc. Se surgirem novas tecnologias, é provável que o Google Docs absorva esta nova maneira de gerar conteúdo.

Sendo assim. Será que posso conectar um blog a Google Docs?

Surgiu o projeto Google Blogs.

Como funciona?

Funciona assim: é criada uma pasta no Google Drive do blogueiro. Por exemplo: “Meu Blog” - qualquer nome serve. Qualquer Google Doc criado nesta pasta passa a ser uma entrada do blog.

Atualmente, imagens, títulos, links, itálicos, negritos e muitas formatações são transmitidas do Google Doc para o blog.

Como foi feito?

Um código hospedado no Google Script faz a conversão. Ele opera todo dia, de madrugada, nesta primeira versão.

Como o mundo do javascript no servidor é poderoso, mesmo ainda nesta primeira versão o Google Blog consegue:

Revisar o texto

Se o autor pedir nas configurações do blog, o Google Blog revisa o texto automaticamente usando IA.

Criar uma hero image

Todo Google Blog cria uma hero image - aquela imagem que fica no topo, antes de começar o texto do blog - quando não há uma imagem inserida no topo do Google Doc.

Por exemplo, a hero image (imagem do topo) do Google Doc “Um Blog Fácil” foi gerada por IA, automaticamente.

Traduzir

Sempre que o Google Doc contiver a indicação “versão em inglês”, “versão em [qualquer idioma]” em qualquer parte do texto, o blog será gerado nestes outros idiomas, também. Cada idioma tendo seu index separado.

Por exemplo, aqui está o índice para entradas em alemão de meu blog. Aqui temos “Um Blog Fácil” em francês. Tudo gerado automaticamente por IA.

Comprimir imagens

Por que não comprimir as imagens antes de publicá-las? Claro que sim, automaticamente. As imagens são comprimidas antes de irem para o blog.

Liberdade simples

E assim consegui liberdade para criar entradas em meu blog, com a simplicidade de uma das ferramentas mais fáceis para gerar conteúdo: um Google Doc.

“Google Blogs”, apelidei.

Tudo é feito a partir de uma pasta do meu Google Drive, que contém meus Google Docs.

Simples assim.

Para mim, hoje, esta é a maneira mais simples de gerar um blog. Sem as restrições que encontrei nas primeiras opções, como Medium e Meta.

Bons textos e boas leituras a todos!

Obs.: notaram que aproveitei a mesma tecnologia para meus projetos? Eles também usam a mesma ferramenta que cria blogs. Assim, a postagem sobre este projeto também não passa de um Google Blog: um Google Doc mais o poder do javascript no servidor.
